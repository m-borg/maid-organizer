A lightweight Chrome extension that keeps your Downloads folder from becoming a dumpster fire. It automatically sorts incoming files into specific folders based on where you downloaded them from or what type of file they are.

<img src="https://lh3.googleusercontent.com/xn5UHg4bmZhsh2knsCq3d4JDfCbYrPtGDJkGtLFZTRlCqyBz1wJvvPs4IcJZ1JziPfOQaF4rL3A3viYoNyU9jo6u3f8=s1600-w1600-h1000" width="600" alt="Maid UI Screenshot">

## What it does

* **Sort by Website:** Send all files from `x.com` to a "Twitter" folder, or school portal files to "Homework".
* **Sort by File Type:** Automatically route `.mp3` files to "Music", or `.jpg`s to "Images".
* **Set & Forget:** Runs invisibly in the background. No manual dragging and dropping.
* **100% Private:** No tracking, no analytics, no external code. It just checks the incoming file and moves it.
